# Demo_Weather Agent
To test this demo, you need to apply for an API from https://bailian.console.aliyun.com, activate the agent by running Weather_Agent.py

## Name
Weather Agent

## Description
This is a demo of a weather forcasting Agent with background resourses needed to handle some of a few specialized questions in Meteorology.

## Installation
You need to install python, Langchain, and FAISS

## Usage
Test reactions of the Agent with inputs like asking the weather, and some background knowledge in Meteorology
